# Planejador Financeiro em Python

Aplicação de terminal desenvolvida em Python para simulação de investimentos e planejamento financeiro, com exportação de dados em CSV.

## Funcionalidades

- Simulação por período fixo  
- Cálculo de meta financeira  
- Exportação dos resultados em arquivo `.csv`  

## Tecnologias

- Python  
- CSV (biblioteca padrão)  

## Estrutura

- main.py  
- Funções de cálculo  
- Função de exportação CSV  
- Menu interativo  

## Objetivo

Projeto desenvolvido para praticar lógica de programação, funções, tratamento de erros e manipulação de arquivos.

## Como executar
Executar via prompt
